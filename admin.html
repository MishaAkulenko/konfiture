Данные для построения страницы передать в виде

    [{
        title: 'Свадьбы на природе',
        chapter: 'Новости',
        poster: 'images/news/news3.jpg',
        link: 'watch?v=xyCdd9qoiKY'
    },
    {
        title: 'Свадьбы на природе',
        chapter: 'Новости',
        poster: 'images/news/news3.jpg',
        link: 'watch?v=xyCdd9qoiKY'
    } ]

Где title - вводимый заголовок для новости, chapter - раздел(может быть либо "Новости" либо "Тренды и советы"), poster - ссылка на постер к новости, link - ссылка на полную статью

Аякс запрос буду делать в виде:

    $.ajax({

        bildNews(mainNews);
